## Step 2.5: Block Inventory

### LOCAL BLOCKS (Already Available):

- **hero**: Large heading, text, and buttons at top of page (empty file - needs implementation)
- **cards**: Grid of items with images, headings, and descriptions
- **columns**: Side-by-side content in 2-3 columns
- **fragment**: Reusable content section that can be embedded across pages
- **header**: Navigation header (auto-populated)
- **footer**: Page footer (auto-populated)

### BLOCK COLLECTION (Can Be Added):

- **accordion**: Expandable sections for FAQs or collapsed content
  - Live example: https://main--aem-block-collection--adobe.aem.live/block-collection/accordion

- **tabs**: Tabbed interface for organizing related content
  - Live example: https://main--aem-block-collection--adobe.aem.live/block-collection/tabs

- **carousel**: Rotating slideshow of images or content
  - Live example: https://main--aem-block-collection--adobe.aem.live/block-collection/carousel

- **quote**: Highlighted testimonial or pullquote with attribution
  - Live example: https://main--aem-block-collection--adobe.aem.live/block-collection/quote

## Summary

The Decathlon homepage has **13 sections** with a total of **20 content sequences** across all sections. The page is primarily composed of:

- Promotional/hero banners (multiple sections)
- Product carousels with horizontal scrolling
- Product grids with detailed information
- Category navigation
- Newsletter signup
- Service highlights

The project already has essential blocks (cards, columns, fragment) and can leverage additional Block Collection blocks (carousel, accordion, tabs) as needed.
